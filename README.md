# Chloe Wang — Portfolio

Personal portfolio site: editorial-style grid, project thumbnails, and light JavaScript for section switching.

## Stack

- Static HTML
- CSS (custom properties, CSS Grid)
- Vanilla JS (`js/main.js`)

## Project layout

| Path | Role |
|------|------|
| `index.html` | Page structure, work / other / about panels |
| `css/style.css` | Layout, grid, stars, responsive breakpoints |
| `js/main.js` | Nav panel toggle, project hover ↔ star highlight |
| `assets/images/projects/` | Project and UI images |

## Run locally

Open `index.html` in a browser, or from this folder:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.

## GitHub Pages（可选）

Repository → **Settings** → **Pages** → Source: **Deploy from a branch** → Branch `main` / folder `/ (root)`.
